# GENFLIX README

# Installation
## Environnement
Dans le dossier de l'application, ouvrir un terminal puis :

1. Créer l'environnement virtuel   ```python -m venv venv ```
2. Activer l'environnement   ``` ./venv/Scripts/activate ```
3. Installer les dépendances  ```pip install -r ./requirement.txt```

## Clé API 

1. Créer une clé API sur Gemini : : https://aistudio.google.com/api-keys
2. Copier la clé API  
3. Coller la clé dans le fichier : .env.template
4. Renommer le fichier : .env.template --> .env


## Base de données 

Exécuter le fichier suivant pour créer la base de données : ``` instance/script.py```


# Lancer l'application 

``` python app.py ```
